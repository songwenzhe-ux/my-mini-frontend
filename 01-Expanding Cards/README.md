# 01 - Expanding Cards

## 概要
クリックされたカードが滑らかに広がるインタラクティブなUI効果です。

## 機能
- カードをクリックすると、そのカードが拡大し、他のカードが縮小する。

## 学んだこと
- CSS 
 - transitionプロパティの柔軟な設定により、カードが展開,縮小する際の滑らかなアニメーション効果を実現します。
- Flexbox
 - `flex: 0.5` から `flex: 5` への切り替えによるレイアウト制御の習得。
- 要素の配置
  - `relative`（親要素）と `absolute`（子要素）を組み合わせた手法。
- JavaScript / DOM操作
  - `querySelectorAll`で取得した要素リストに対して、`forEach`構文を使用して繰り返し処理を行う基本的な使い方を習得しました。
  - `forEach`を利用し、複数のパネルにクリックイベントを一括で設定する処理を実装しました。
