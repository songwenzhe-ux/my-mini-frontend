# 02 - Rotate Sidebar Menu

## 概要
クリックで画面全体が回転し、サイドバーメニューがスライド表示されるインタラクティブ UI です。

## 機能
- OPEN ボタンをクリックすると画面が回転し、サイドナビがスライドインする。
- CLOSE ボタンをクリックすると元の位置に戻り、メニューが非表示になる。

## 学んだこと
- CSS
  - transform: translateX () と transition を組み合わせて、スライドアニメーションを実現する方法を習得しました。
  - width と max-height の違いと使い分けを理解しました。
  - list-style-type でリストの装飾を削除する方法を学びました。
  - 隣接セレクター（li + li）を使って、特定の要素にスタイルを適用する方法を習得しました。
- JavaScript
  - classList.add/removeでクラスを切り替え、UIの状態を管理する基本を学びました。
